The Quest For 3%: Strategies to Improve H0 Precision with the Megamaser Cosmology Project

Location: National Radio Astronomy Observatory (Charlottesville, VA)

Abstract: The Megamaser Cosmology Project (MCP) aims to measure the Hubble Constant (H0) with high precision using water megamasers in the disks of nearby active galaxies. The current measurement of H0 using this method is 73.9 ± 3.0 km s-1 Mpc-1, corresponding to an uncertainty of 4.06%. This study explores whether achieving that level of precision is feasible by using existing observational facilities...
